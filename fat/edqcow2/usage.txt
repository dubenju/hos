edqcow2 <image_file> create [-f format] <size> -cs cluster_size
用指定的格式创建指定大小的镜像文件。
image_file 镜像文件名
format     镜像文件的格式，qcow2
size       镜像文件的大小，默认单位KB，可以指定 MB GB TB。

cluster_size

创建头信息1簇
创建表1，(((total_size * 512) + (1LL << shift) - 1) >> shift) 个表项*sizeof(uint64_t) 大小相当的簇;
创建引用计数表
创建引用块表

创建引用块表的大小计算

edqcow2 image_file fdisk create PrimaryPartition [-f format] size
创建表2，更新表1.
更新引用计数块
创建指定大小的主分区
创建这个分区的VBR，值为F6
创建MBR的分区表

edqcow2 image_file fdisk create ExtendedPartition [-f format] size
创建指定大小的主分区
更新前一个MBR的分区表
创建这个分区的EBR
-- 创建这个分区的VBR，值为F6

edqcow2 image_file fdisk create LogicalPartition [-f format] size
更新这个分区的EBR
创建这个分区的VBR，值为F6

edqcow2 image_file fdisk delete PartitionNo

edqcow2 image_file info

edqcow3 image_file copy /dev1/path/file -> c:/path/file
edqcow3 image_file copy c:/path/file -> /dev1/path/file 

edqcow3 image_file delete /dev1/path/file
